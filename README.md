Real-Time Collaborative Notes App

A real-time note-taking application where multiple users can write and edit the same note simultaneously — powered by WebSockets (Socket.IO) and built with the MERN stack.

Tech Stack

- MongoDB– database for storing notes
- Express.js – backend server and REST API
- React – frontend interface
- Node.js – server runtime
- Socket.IO – real-time communication
- Axios– HTTP requests

Features

-  Create and edit notes
-  Live collaboration across multiple tabs/devices
-  Auto-save every few seconds
-  Display active collaborators in a room
- Show last updated time for note
